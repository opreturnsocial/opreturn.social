# opreturn.social

> _Freedom is not free_

The first official [ORS](https://github.com/opreturnsocial/ors) client (web + CLI).

**[Try it now](https://opreturn.social)**

This client includes a cache and lightning payments facilitator to simplify UX, and a hybrid free/paid model using Mutinynet & Mainnet bitcoin to reduce costs and lower the initial onboarding barrier.
